# AI Game Studio - 前端办公室场景与交互设计文档

## 1. 目标
前端需要同时展示空间、流程、协作三类信息，而不是普通后台表格。

## 2. 技术建议
- React
- TypeScript
- Vite
- Zustand
- Framer Motion
- SVG / CSS 渲染

## 3. 总体布局
### 左侧：办公室空间视图
- 5个角色
- 唯一电脑
- 队列区
- 气泡
- 状态标签

### 中间：小黑板
- 当前阶段
- 任务列表
- owner
- status
- 通过/拒绝按钮

### 右侧：留言板与日志
- 用户与制作人对话
- Agent 间关键对话
- 系统事件日志
- 代码产出摘要

### 顶部栏
- 项目名
- 当前阶段
- 系统状态
- 在线角色数

## 4. 风格要求
- 深色背景
- 霓虹边缘
- 轻发光
- 科技感
- 清晰信息层级

## 5. 用户关键操作
- 提交需求
- 回答制作人问题
- 批准阶段计划
- 驳回阶段计划
- 查看代码产出摘要
- 查看测试报告

## 6. 聚合接口
建议使用 GET /api/scene 返回：
- agents
- computer_lock
- blackboard
- latest_messages
- event_logs
- current_phase
- artifacts_summary

## 7. 成功验收
用户能直观看到：
1. 办公室里的角色正在协作
2. 谁在占用电脑
3. 目前做到哪一阶段
4. 最近发生了什么
5. 我该在哪里批准下一步
